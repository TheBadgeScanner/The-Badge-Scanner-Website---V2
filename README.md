
  # Albert - The Badge Scanner Website - Version 2
  
  This project is deployed to **GitHub Pages** from the `docs/` folder on the `main` branch.

  URL: https://thebadgescanner.github.io/The-Badge-Scanner-Website---V2/

  After making changes/updates, easily deploy with the below in the terminal to see the latest changes on the website:

  # npm run deploy


  ## Running the code locally

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
